# Shape Survivor

Shape Survivor is an offline-first HTML5 canvas survival defense game packaged for iOS with Capacitor.

## Current Features

- One-thumb arena survival gameplay
- Left/right thumb control setting
- Mute setting for audio and haptics
- Offline missions and local achievements
- Unlockable core classes
- Boss relic chest rewards
- Powerups, elite enemies, bosses, and random run events
- PWA/offline support through a web manifest and service worker

## Development

```sh
npm test
npm run sync
```

`npm run sync` copies the source web bundle into `www` and then into the iOS Capacitor public bundle.
